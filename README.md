This project creates a smart infrared remote control capable of triggering custom scenes, such as turning on a TV or executing a sequence of remote-control commands.

The system reads infrared signals from an existing remote control, saves each button code, and allows the user to program scenes that repeat those commands when triggered. Each scene can be activated through local HTTP routes, making it easy to integrate with future automation systems, voice commands, smart assistants, and IoT platforms.

The main goal of this project is to support accessibility, home automation, and IoT solutions by allowing devices controlled by traditional infrared remotes to be operated through simple digital commands.
